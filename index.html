Community Hub

Welcome to the Genesis Community Hub, where you'll find the latest on Genesis and Genesis Worlds

The Genesis Community Hub contains the source of truth for all things Genesis Community, including Governance, community standards, and the Community Improvement Proposals (CIPs) process.

CIPs

CIPs describe standards for the Genesis platform, including core protocol specifications, client APIs, and contract standards. To review and contribute to the CIPs, go to the Community Improvement Proposal Hub

Contributing

First review CIP-1. Then clone the repository and add your CIP to it. There is a template CIP here. Then submit a Pull Request to Genesis's CIPs repository.
